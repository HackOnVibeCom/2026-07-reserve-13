# Tact - demo script (60-90s)

## Setup

- Browser: landing `/` then desk `/app`
- Prefer live mode if `OPENROUTER_API_KEY` is set; otherwise use Demo drafts only

## Voiceover

1. **Hook (10s)**  
   “You just launched a mobile app. Free promotion means Reddit and Indie Hackers - but one spammy post and you’re done.”

2. **Product (10s)**  
   “This is Tact. It writes community-native posts for a newly launched app and scores spam risk with real rules.”

3. **Flow (40s)**  
 - Click **Load sample: Focusrail**  
 - Select **r/SideProject**  
 - Click **Draft with Tact**  
 - Point at **Spam posture** (high risk, CTA-first)  
 - Point at **Soft draft** (problem-first, one ask, link late)  
 - Scroll **findings** - CTA-first, hard-sell, etc.

4. **Business (15s)**  
   “Buyer is a solo indie in week one. Free tier for a few drafts, Pro at nine dollars a month. We’d dogfood Tact to promote Tact.”

5. **Close (5s)**  
   “Promote the launch. Without sounding like an ad.”

## Claims to avoid on camera

- Do not say “guaranteed virality” or “algorithm hack”
- Do not claim auto-posting to Reddit
- Risk score is **rules-based**, not a platform moderation API
