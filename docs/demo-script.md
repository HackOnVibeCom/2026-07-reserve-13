# Tact demo script (75–100s)

Record in 1080p. Dark or light theme is fine; **dark looks cleaner on camera**.

## Setup (before record)

1. Incognito or cleared local history (clean desk).
2. Open https://tact-liard.vercel.app then https://tact-liard.vercel.app/app
3. Theme: **Dark**
4. Prefer **Demo drafts only** for a deterministic take (or live if key is warm).
5. Cursor large enough to see clicks.

## Shot list + voiceover

### 0:00–0:08 · Hook (landing)

Show landing hero: “Promote the launch. Without sounding like an ad.”

> “You just shipped. Free promo means Reddit, Product Hunt, X. One spammy post and the room ignores you.”

Click **Open the desk**.

### 0:08–0:18 · Product (desk empty)

> “This is Tact. Soft community posts for a newly launched product, scored by a real risk engine, not vibes.”

### 0:18–0:55 · Core flow

1. Click **Sample: Prooflet** (or Focusrail if you want pure mobile story).
2. Select **r/SideProject**.
3. Click **Demo drafts only** (or Draft with Tact).
4. Point at **risk before → after** meters (spam high → soft low).
5. Point at **Spam posture** (download-now energy).
6. Point at **Soft draft** (problem, wedge, one ask, link late).
7. Briefly show findings list.

> “Same product. Two postures. Rules catch CTA-first, hard sell, early links, missing problem, missing ask.”

### 0:55–1:15 · Depth

1. Edit one line in the soft draft (change a word).
2. Click **Re-score risk**.
3. Click **share** icon (no need to read the URL aloud).
4. Optional: flip to **Product Hunt maker** or **X**, demo draft once.

> “Edit in place, re-score, share a judge link. Three free live drafts per browser; demo is unlimited.”

### 1:15–1:30 · Business + close

> “Buyer is a solo builder in week one. Free live drafts, unlimited demo, nine dollars a month if we keep building. Promote the launch without sounding like an ad.”

End on soft draft + meters. Freeze 1s.
